== Installation ==



1. Upload `wpbatch-gallery-slideshow.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see a new menu in the dashboard "WPBatch Gallery"... add some images from there
4. now just add this shortcode anywhere in the page, post or widgets 
<pre>[wpbatch-gallery]</pre>
or Place `<?php do_shortcode('[wpbatch-gallery]'); ?>` in your templates php files

Good Luck
